OpENer Code.

This is an open source (BSD) implementation of the ODVA (www.odva.org) CIP Ethernet/IP industrial automation protocol stack.

  • EtherNet/IP(TM) stack for I/O adapter devices
  • supports multiple I/O and explicit connections;

This originally was a git svn clone from http://sourceforge.net/projects/opener/ which disappeared from Sourceforge after I cloned it. I will maintain this as a fork.

Another active maintainer is here https://github.com/CapXilinx/OpENer/, which I will track alongside any other implementations I can find.
